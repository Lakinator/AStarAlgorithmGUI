# AStarAlgorithmGUI
Visualization of the A* Algorithm, based on https://github.com/Lakinator/AStarAlgorithm
> Note: The "compiled" folder is for windows only and is bundled with necessary gtk dll's

## About

- Tiles:
  - Erase: White
  - Wall: Black
  - Start: Red
  - End: Green
  - Path: Yellow
- Slider:
  - Changes size of the cells
